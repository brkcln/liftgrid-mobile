# LiftGrid Saha — Mobile App Releases

Signed Android APK releases for **LiftGrid Saha**, the field technician app for
[LiftGrid](https://getliftgrid.com) elevator maintenance management.

## Download

- **Latest release:** [liftgrid-saha.apk](https://github.com/brkcln/liftgrid-mobile/releases/latest/download/liftgrid-saha.apk)
- **All releases:** [Releases tab](https://github.com/brkcln/liftgrid-mobile/releases)

## Installing

1. Download the APK on your Android device.
2. When prompted, allow your browser to install apps from unknown sources.
3. Open the downloaded file and confirm install.
4. Sign in with the credentials provided by your LiftGrid administrator.

Requires Android 8.0 (API 26) or newer.

## What this repo is

This repository hosts **only** the signed APK binaries cut from the private
[`brkcln/sektorpanel`](https://github.com/brkcln/sektorpanel) monorepo. Source
code is not published here. The repo exists so end-user devices can fetch
updates from a stable public URL without authentication.

In-app update notifications are driven by the version-check endpoint at
`api.getliftgrid.com/api/mobile/version`.
